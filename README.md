# Synth

Projekt syntezatora audio w Unity, wykorzystujący syntezę subtraktywną i sekwencer krokowy.

## Funkcje

- Różne waveformy (Sine, Square, Saw, Triangle, Noise, Sine2, SoftSquare, SoftSaw, Sine3, SoftPulse)
- Sekwencer krokowy z możliwością akcentowania nut
- Kontrola parametrów dźwięku (głośność, wzmocnienie, akcenty)
- Interakcja z obiektami 3D (ruch i siła)

## Wymagania

- Unity 2022.3 lub nowsza
- .NET 4.x

## Instalacja

1. Sklonuj repozytorium
2. Otwórz projekt w Unity
3. Otwórz scenę z Assets/Scenes

## Użycie

1. Dodaj komponent VCO do obiektu
2. Skonfiguruj parametry dźwięku w inspektorze
3. Ustaw kroki w sekwencerze
4. Uruchom grę, aby usłyszeć dźwięk
